# SnowFox v1.3.5

发布日期：2026-03-20

本次版本主要聚焦三件事：批量规划体验、打印流程可用性，以及计划校验与发布链路的一致性整理。对于当前仍停留在 `1.3.4` 的用户，建议升级到 `1.3.5`。

## 下载

- Windows 安装包：`https://snowfox-release.oss-cn-beijing.aliyuncs.com/SnowFox-Setup-1.3.5.exe`

## 本次更新

### 新增

- Overview 视图支持空位多选，批量新增样本时可以一次规划多个目标位置。
- 设置页支持调整自定义字段顺序，操作表单和上下文展示会按 schema 顺序保持一致。
- 网站历史版本 list 现在可以展示版本发布日期，并在悬停时显示该版本摘要。

### 改进

- 操作单打印流程改为系统打印预览对话框，不再依赖先导出 HTML 再交给浏览器打开。
- 打印版操作单网格样式重新整理，占位标签显示更完整，边框、标记和浅色打印效果更清晰。
- Overview 网格、表格状态色和行内图标进一步统一到同一套主题视觉语言。
- AI 会话运行时从 GUI bridge 中拆出独立 session service，设置变更后的 API Key 同步路径更明确。
- 发布链路统一回收到了 `app_gui/version.py`、`latest.json`、`CHANGELOG.md` 和安装器版本这几处明确入口，后续发版更容易校对。

### 修复

- 计划合并前会更早识别 “add 覆盖 add” 冲突，减少批量执行时的晚发现错误。
- 计划校验入口下沉到共享核心后，GUI 与 Agent 对同类计划的规则来源保持一致。
- 设置保存后，当前 AI 会话会立即刷新 API Key 配置，避免继续沿用旧会话参数。

## 升级说明

- 自动更新元数据已经同步到 `1.3.5`。
- 如果你使用的是 Windows 安装包升级方式，建议先关闭正在运行的 SnowFox，再执行安装器覆盖安装。
- 若你的下载链路启用了防盗链，请确保应用更新检查来源和下载来源都在允许范围内。
