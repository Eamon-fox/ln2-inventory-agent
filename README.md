> 给生医药实验室的液氮冻存盒管理软件：像 Excel 一样好上手，但比 Excel 更不容易出错。

👉 安装包下载（Releases）：https://github.com/Eamon-fox/snowfox/releases

我是做湿实验出身，按真实液氮场景设计这个工具：
时间短、手会冻、不能反复翻找、登记必须快且准。

## 一句话定位

- 比 Excel 更稳，比 LIMS 更轻
- 面向研究生课题组，也适合小型企业研发团队
- 默认 GUI 可直接用，AI Agent 是可选加速器（不是必须）

## 为什么它更适合液氮场景

- **先查后取**：上罐前先定位，减少开罐后翻找时间
- **9x9 可视化**：盒位一眼看懂，避免“看花眼”
- **列表 + 盒位双视图**：保留 Excel 习惯，同时更智能
- **一键操作清单**：可抄写/可打印，带单去液氮罐更从容

## 核心功能

- 一张 `YAML` 走天下，结构清晰，适合长期维护
- 随时导出 `CSV`，满足“落袋为安”的备份习惯
- 支持字段分类与颜色标记，按各实验室习惯自定义
- 适配不同液氮罐尺寸和索引方式
- 智能校验防错（如同一格位冲突）
- 全程操作留痕，可回滚到任意历史记录

## 快速开始

- 直接在 Releases 下载并安装对应系统的安装包
- 安装后双击打开即可使用，无需命令行
- 可选启用 AI Agent：支持 Zhipu 和 DeepSeek，自己带 key 配好后直接启动即可
- AI 读功能全面且迅速；所有写操作都需要人类审核后才会执行，保证安全可靠

## 典型使用流程（先查后取）

1. 在电脑上搜索样本并确认盒位
2. 一键生成操作清单（抄写或打印）
3. 到液氮罐前按清单快速取/存
4. 操作后立即回填，保持库存与现实同步
