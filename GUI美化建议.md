# SnowFox GUI 美化建议（仅前端视觉与交互）

> 目标：不改底层逻辑/后端，仅提升界面观感与操作体验。

## 一、品牌调性

- 统一主色体系：建议围绕“冰川蓝 + 雪白 + 深海灰”建立 light/dark 一致的品牌色。
- 保持 SnowFox 品牌一致：窗口图标、标题文案、关于页链接统一为 SnowFox。
- 可选：增加简洁启动画面（Splash）强化品牌记忆。

## 二、布局与信息层级

- 优化三栏分隔（Overview / Operations / AI）：增强分割线可见性与拖拽反馈。
- 提升卡片化层级：筛选区、统计区、操作区使用统一卡片边界与间距。
- 降低视觉噪声：减少重复边框，重点区域（当前操作、选中状态）更突出。

## 三、组件样式优化

- 表格/列表：统一行高、hover、选中态、表头层级，提升可读性。
- 冻存盒 9x9 视图：加强选中格位、预览态（add/takeout/move）的对比度与一致性。
- 按钮体系：主按钮、危险按钮、次级按钮、幽灵按钮语义更清晰。
- 输入控件：focus 态更明显，禁用态对比更柔和。

## 四、交互体验

- 增加加载反馈：耗时操作显示 spinner/状态文案，避免“无响应”感。
- 增加空状态设计：列表为空时给出引导文案与下一步建议。
- 优化微交互：面板切换/展开收起加入轻量过渡（短时长，不影响效率）。

## 五、可访问性与一致性

- 对比度检查：确保浅色/深色主题文本可读。
- 字号与间距基线统一：标题、正文、注释保持稳定节奏。
- 中英文混排优化：重点检查中文场景的字体回退和截断。

## 六、推荐实施顺序

1. 品牌色与全局 token 统一（低风险，高收益）
2. 三栏布局与卡片层级优化（建立结构感）
3. 9x9 视图与列表样式统一（核心高频区域）
4. 加载态与空状态补齐（体验完善）
5. 动效与细节打磨（最后加分项）

## 七、验收标准（UI 侧）

- 首屏 3 秒内可识别主要操作入口。
- “先查后取”路径操作无视觉歧义。
- light/dark 主题均可读、可辨、可快速定位。
- 不改变业务逻辑与数据行为。
